# Python White Board Application

This project is a **simple whiteboard drawing application** built with **Python and Tkinter**.  
It allows users to draw freely on a canvas, change colors, adjust brush size, and clear the board.

The main purpose of this project is to demonstrate **basic Python usage**, GUI development, and event-driven logic.
